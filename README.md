# Sports-Radio-Sentiment-Analysis
Transcribed Boston sports radio episodes and performed a sentiment analysis in R on each speaker and date, and used these variables to track patterns with sentiment. 

See Google Slides presentation for more details: https://docs.google.com/presentation/d/1xjGhp3JDDZMZabGaoGQ8Hc_oO6xg0xs3fsOuu4tWE3A/edit?slide=id.p#slide=id.p

Mapping Texts: From Text To Numbers by Stoltz and Taylor used for analysis technque ideas
